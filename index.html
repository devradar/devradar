Competence management
for developers

Manage your competencies as an individual using the devradar editor or track competence management within your team by using both the editor and the devradar teams app.

  • Manage competencies via web UI
  • View all skills in a list or radar view
  • Import/export via .toml files
  • Create shareable url devradar.io/@yourname
  • Login with Twitter/GitHub/Google Account
  • (soon) Overlay your team’s devradars and focus on the key skills
